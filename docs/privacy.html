Privacy Policy

Last updated: February 2026

Boards & Pins MCP Server is a personal productivity tool. It does not collect, sell, or share your personal data. All data stays between your device and Pinterest's official API.

1. What is this app?

Boards & Pins MCP Server is an open-source tool that connects Claude AI (by Anthropic) to the Pinterest API v5 via the Model Context Protocol (MCP). It allows users to browse, create, and manage their own Pinterest boards and pins through a conversational interface.

It is not a commercial product. It is developed and maintained by Anne Sapyx for personal use.

2. Data we collect

The only data stored by this app are the OAuth authentication tokens required to access the Pinterest API on your behalf:

  • Pinterest access token
  • Pinterest refresh token
  • Token expiry timestamps
  • Granted OAuth scopes

These tokens are saved locally on your own machine at ~/.mcp-credentials/pinterest-tokens.json with restricted file permissions (readable only by your user account). They are never uploaded to any server.

3. Data we do NOT collect

This app does not:

  • Collect your name, email address, or any personally identifiable information
  • Store any Pinterest content (pins, boards, images, descriptions)
  • Use cookies or any tracking technologies
  • Send data to any third party other than Pinterest's official API
  • Log or retain your Pinterest activity
  • Run any backend server or database

4. How we use Pinterest data

API calls are made exclusively to https://api.pinterest.com/v5 using your own credentials, on your behalf, in response to your direct requests. No Pinterest data is retained after the API response is shown to you.

5. Data sharing

We do not share any data with third parties. The only external service this app communicates with is Pinterest (Pinterest Privacy Policy).

6. Data storage and security

OAuth tokens are stored locally on your device with 0o600 file permissions (owner read/write only). No data is stored on remote servers. You can delete the token file at any time by removing ~/.mcp-credentials/pinterest-tokens.json.

7. Your rights

Since no personal data is collected or stored by us, there is no data to access, correct, or delete on our end. To revoke this app's access to your Pinterest account:

  • Go to your Pinterest account settings
  • Navigate to Security & privacy → Apps with access
  • Remove Pinterest MCP Server

8. Children's privacy

This app is not directed at children under 13. We do not knowingly collect any information from children.

9. Changes to this policy

If this policy is updated, the new version will be published at this URL with a revised date.
