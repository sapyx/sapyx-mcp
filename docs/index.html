14 MCP Tools

Every tool is validated with Zod and returns structured content blocks.

Auth

  • pinterest_authStart the OAuth 2.0 browser flow
  • pinterest_auth_statusCheck token validity and expiry

Boards

  • list_boardsList all boards for the authenticated user
  • create_boardCreate a new board with optional privacy setting
  • list_board_sectionsList sections within a board
  • create_board_sectionAdd a new section to an existing board

Pins

  • list_pinsList pins on a board or section
  • get_pinGet full details and image URLs for a pin
  • get_pin_imageFetch a pin image as a viewable base64 block
  • update_pinEdit title, description, alt text, or link
  • move_pinMove a pin to a different board or section
  • create_pinCreate a new pin from a public image URL

Search & User

  • search_pinsSearch pins by keyword with pagination
  • get_user_profileRetrieve the authenticated user's profile

Installation

Requires Node.js 20+.

# Clone and build
git clone https://github.com/sapyx/sapyx-mcp.git
cd sapyx-mcp
npm install
npm run build

# Copy env template and add your credentials
cp .env.example .env

Authentication

Choose one mode. Both cannot be active simultaneously.

Direct Token

Read-only

Generate a static token from the Pinterest Developer Portal. No browser interaction needed.

PINTEREST_ACCESS_TOKEN=pina_...

OAuth 2.0

Full access

Full read + write access with automatic token refresh. Trigger via the pinterest_auth tool.

PINTEREST_APP_ID=...
PINTEREST_APP_SECRET=...
